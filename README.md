Analyzed Netflix dataset using Python, Pandas and Matplotlib to uncover trends in content type, genres, release years, and country-wise distribution, with data cleaning, EDA, and visual storytelling.

# 🎬 Netflix Data Analysis Project

## 📌 Description
The Netflix Data Analysis Project focuses on exploring and analyzing Netflix's content dataset to uncover meaningful insights. Using Python and data visualization tools, this project highlights trends in movies and TV shows based on genre, release year, country, and ratings. It includes data cleaning, exploratory data analysis (EDA), and visual storytelling to better understand the platform's content distribution.

---

## 📂 Dataset
- Source: Netflix dataset (CSV file)
- Features include:
  - Title
  - Type (Movie / TV Show)
  - Director
  - Cast
  - Country
  - Release Year
  - Rating
  - Duration
  - Genre (listed_in)

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔍 Project Workflow

### 1. Data Cleaning
- Handling missing values
- Removing duplicates
- Formatting columns

### 2. Exploratory Data Analysis (EDA)
- Movies vs TV Shows distribution
- Year-wise content growth
- Top genres on Netflix
- Country-wise content analysis
- Ratings distribution

### 3. Data Visualization
- Bar plots
- Count plots
- Heatmaps
- Trend analysis

---

## 📊 Key Insights
- Netflix has more Movies than TV Shows
- Rapid increase in content after 2015
- Majority of content is produced in the United States
- Drama and Comedy are the most popular genres
- TV Shows often have multiple seasons
