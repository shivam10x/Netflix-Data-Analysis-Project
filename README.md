# Netflix-Data-Analysis-Project
Analyzed Netflix dataset using Python, Pandas and Matplotlib to uncover trends in content type, genres, release years, and country-wise distribution, with data cleaning, EDA, and visual storytelling.

📌 Overview

This project focuses on analyzing Netflix content data to uncover insights about movies and TV shows available on the platform. Using Python and data visualization libraries, we explore trends in content type, genres, release years, ratings, and more.

📂 Dataset
Source: Netflix dataset (CSV format)
Contains information such as:
Title
Type (Movie/TV Show)
Director
Cast
Country
Release Year
Rating
Duration
Genre (Listed in “listed_in” column)
🛠️ Tools & Technologies
Python 🐍
Pandas
NumPy
Matplotlib
Jupyter Notebook
🔍 Key Analysis Performed
1. Data Cleaning
Handled missing values
Converted data types
Removed duplicates
2. Exploratory Data Analysis (EDA)
Distribution of Movies vs TV Shows
Content added over the years
Top genres on Netflix
Country-wise content production
Most frequent ratings
3. Data Visualization
Bar charts
Count plots
Heatmaps
Time-series trends
📊 Key Insights
Netflix has more Movies than TV Shows
Significant growth in content after 2015
Most content comes from the United States
Popular genres include Drama, Comedy, and Documentary
TV Shows tend to have multiple seasons
