# Netflix-Data-Analysis-Project
Analyzed Netflix dataset using Python, Pandas and Matplotlib to uncover trends in content type, genres, release years, and country-wise distribution, with data cleaning, EDA, and visual storytelling.



## 📌 Overview

This project analyzes Netflix's dataset to uncover insights about movies and TV shows available on the platform. It focuses on identifying content trends, popular genres, and distribution across different countries using Python and data visualization techniques.

---

## 📊 Objectives

* Analyze the distribution of Movies vs TV Shows
* Identify trends in content release over the years
* Find the most popular genres
* Explore country-wise content production
* Understand ratings and duration patterns

---

## 📂 Dataset

Dataset used: netflix_titles.csv
Source: Public Netflix dataset (Kaggle / open source)

Key columns:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre (listed_in)

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

---

## 🔍 Project Steps

### 1. Data Cleaning

* Handling missing values
* Removing duplicate records
* Converting data types

### 2. Exploratory Data Analysis (EDA)

* Movies vs TV Shows comparison
* Year-wise content growth
* Genre distribution
* Country-based analysis
* Ratings analysis

### 3. Data Visualization

* Bar charts
* Count plots
* Heatmaps
* Trend graphs

---

## 📈 Key Insights

* Movies dominate Netflix content compared to TV Shows
* Significant growth in content after 2015
* United States contributes the most content
* Drama, Comedy, and Documentaries are top genres
* TV Shows typically have multiple seasons
